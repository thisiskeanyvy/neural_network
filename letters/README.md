## Structure des dossiers et fichiers

```
FontName-Type/
    Contient la liste des lettres actuelles à entraîner (en BMP)

fonts/
    Contient la liste des fonts qui permet de créer les images pour l'entraînement (TTF)

generator.py
    Permet de générer des lettres à partir de certaines fonts
    (modifier la ligne avec le fichier de fonts)
```
